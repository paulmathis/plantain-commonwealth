# Plaintain Commonwealth

## About

The Plaintain Commonwealth is a demo E-Commerce clothing store.

## Stack

- Mongodb
- Express
- React
- Redux
- NodeJS

## How to use

Clone the repository

Install node modules for server and client

```
npm install && cd ./client && npm install && cd ../
```

Run the server

```
npm run dev
```
